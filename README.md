# Todo List with React

- 開發使用[Create React App] (https://github.com/facebook/create-react-app)

## 使用本專案

1. clone 本專案 repo
   `git clone https://github.com/kathyhu91/react-ToDoList.git`

2. 安裝 npm
   ` npm init`

3. 專案啟動
   ` npm start`

4. 瀏覽本專案網頁
   `於瀏覽器打開路徑 http://localhost:3000 `

5. 停止專案執行
   `Ctrl+c`
